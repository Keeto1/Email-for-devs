Quick Start

Get up and running with FAFCODE in minutes

1. Installation

Install the FAFCODE SDK for your preferred language

npm install @fafcode/sdk

2. Configuration

Set up your API key and configure the client

const client = new FafcodeClient({
  apiKey: 'your-api-key'
});

3. Send Email

Send your first email using the API

await client.send({
  to: 'user@example.com',
  subject: 'Hello!'
});

API Reference

Complete API documentation with examples

Authentication

All API requests require authentication using your API key.

Header Authentication

Authorization: Bearer your-api-key

Query Parameter

?api_key=your-api-key

Send transactional emails using our REST API.

Code Examples

See FAFCODE in action with real code examples

JavaScript/Node.js

const FafcodeClient = require('@fafcode/sdk');

const client = new FafcodeClient({
  apiKey: process.env.FAFCODE_API_KEY
});

async function sendWelcomeEmail(user) {
  try {
    const response = await client.emails.send({
      to: user.email,
      subject: 'Welcome to our platform!',
      template: 'welcome',
      data: {
        name: user.name,
        loginUrl: 'https://app.example.com/login'
      }
    });
    
    console.log('Email sent:', response.id);
  } catch (error) {
    console.error('Failed to send email:', error);
  }
}

Python

from fafcode import FafcodeClient

client = FafcodeClient(api_key="your-api-key")

def send_welcome_email(user):
    try:
        response = client.emails.send(
            to=user.email,
            subject="Welcome to our platform!",
            template="welcome",
            data={
                "name": user.name,
                "login_url": "https://app.example.com/login"
            }
        )
        print(f"Email sent: {response.id}")
    except Exception as e:
        print(f"Failed to send email: {e}")
