Quick Start

1. Get Your API Key

Sign up for a free account and get your API key from the dashboard.

Terminal 
curl -X GET https://api.fafcode.com/auth/key \
  -H "Authorization: Bearer YOUR_TOKEN"

2. Send Your First Email

Use our simple API to send your first email in seconds.

JavaScript 
const fafcode = new FafcodeClient('YOUR_API_KEY');

const result = await fafcode.emails.send({
  to: 'user@example.com',
  from: 'noreply@yourapp.com',
  subject: 'Welcome to our app!',
  html: '<h1>Welcome!</h1><p>Thanks for signing up.</p>'
});

console.log('Email sent:', result.id);

3. Track Your Email

API Endpoints

POST /v1/emails/send

Send Email

Send a single email to one or more recipients.

Parameters

to required

Email address or array of email addresses

from required

Sender email address

subject required

Email subject line

html optional

HTML content of the email

Example Request 
curl -X POST https://api.fafcode.com/v1/emails/send \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to": ["user@example.com"],
    "from": "noreply@yourapp.com",
    "subject": "Welcome to FAFCODE",
    "html": "<h1>Welcome!</h1><p>Thank you for joining us.</p>"
  }'

Code Examples

Welcome Email Example

import { FafcodeClient } from '@fafcode/sdk';

const client = new FafcodeClient(process.env.FAFCODE_API_KEY);

async function sendWelcomeEmail(userEmail, userName) {
  try {
    const result = await client.emails.send({
      to: userEmail,
      from: 'welcome@yourapp.com',
      subject: `Welcome to Our Platform, ${userName}!`,
      html: `
        <div style="font-family: Arial, sans-serif; max-width: 600px; margin: 0 auto;">
          <h1 style="color: #e99b63;">Welcome, ${userName}!</h1>
          <p>Thanks for joining our platform. We're excited to have you on board!</p>
          <a href="https://yourapp.com/dashboard" 
             style="background: #e99b63; color: white; padding: 12px 24px; 
                    text-decoration: none; border-radius: 6px; display: inline-block;">
            Get Started
          </a>
        </div>
      `,
      tags: ['welcome', 'onboarding'],
      track_opens: true,
      track_clicks: true
    });
    
    console.log('Welcome email sent:', result.id);
    return result;
  } catch (error) {
    console.error('Failed to send welcome email:', error);
    throw error;
  }
}
